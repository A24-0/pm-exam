```Курсовой проект по .NET
Менеджер паролей
Краткое описание проекта
	Менеджер паролей — это инновационное приложение, предназначенное для безопасного и удобного хранения паролей. Оно позволяет пользователям управлять своими учетными 		записями, сохраняя их в зашифрованном виде, что обеспечивает высокий уровень защиты от несанкционированного доступа.

Список фич
Критические функции
	1. Алгоритм шифрования AES-128: Приложение использует протокол шифрования AES-128 для обеспечения безопасности паролей. Это гарантирует, что даже если злоумышленник получит доступ к данным, он не сможет восстановить исходные пароли.
	2. Генератор паролей: Встроенный генератор паролей помогает пользователям создавать надежные пароли, которые легко запоминать, но сложно угадать.
	3. Различный приоритет у паролей: Пользователи могут присваивать разные уровни приоритета своим паролям, что позволяет быстро определить, какие учетные записи требуют немедленного внимания.
Желательные функции
	4. Двухфакторная аутентификация в приложении: Для дополнительной безопасности предлагается реализация двухфакторной аутентификации прямо в приложении. Это значительно повышает уровень защиты, требуя от пользователей предоставления второго фактора аутентификации.
	5. Проверка пароля на надежность: Приложение может включать функцию проверки пароля на надежность, которая анализирует сложность пароля и предлагает улучшения для повышения его безопасности.
Возможные функции
	6. Создание групп паролей: Пользователи могут организовывать свои пароли в группы, что упрощает управление большим количеством учетных записей.
	7. Внедрение облачных хранилищ: Интеграция с облачными хранилищами позволит пользователям синхронизировать свои пароли между различными устройствами и обеспечит доступ к ним в любой точке мира.
	8. Интеграция с браузером: Приложение может предложить интеграцию с популярными веб-браузерами, что позволит автоматически заполнять формы входа с помощью сохраненных паролей.
```
